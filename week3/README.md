# Week 3 - Weather MCP Server

This is a local Model Context Protocol (MCP) server that wraps the free [Open-Meteo API](https://open-meteo.com/) to provide real-time weather data and forecasts. It runs locally using STDIO transport.

## Prerequisites
- Python 3.10+
- `mcp` and `httpx` Python packages.

## Setup and Installation
1. Navigate to your project root.
2. Install the required dependencies using pip or your environment manager:
   ```bash
   pip install mcp httpx
